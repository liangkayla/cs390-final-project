# cs390-final-project
## Repository Overview
- gemm_experiment.go: simulation of parallelizing GEMMs (general matrix multiplications) among multiple GPUs (represented using goroutines).
- results.txt: timings and other metrics from a run of gemm_experiment.go, the same run that the results in the writeup are from.  
- lab directory: instructions, code, and tests for making this experiment into a lab for future distributed systems students.
